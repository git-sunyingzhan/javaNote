1、Java的跨平台
	跨平台：同一个java程序，可以在多个平台（操作系统）上面运行
	原因：每个平台都有对应的JVM（不同平台的JVM不同）
	tips：
		JVM、JRE和JDK
		1）JVM（Java Virtual Machine）
			Java虚拟机（sandbox沙盒）
		2）JRE（Java Runtime Environment）
			Java运行时环境，JRE = 类库+JVM
		3）JDK（Java Development Kit）
			Java开发工具包，JDK = JRE+一些工具
2、HelloWord案例
	一个Java程序中，会包含很多文件。其中存储源代码的文件称为源文件。
	而源码是以类的形式进行呈现。
	一般一个源文件中，只会存放一个类。
	步骤：
		1）定义类
			class 类名{}
		2)在类里面定义main方法
			public static void main(String[] args){}
		3)在main方法内给出一些语句，比如打印语句
			System.out.println("Hello Word");
3、JDK环境变量
	目的：	在任意位置进行编译运行
	JAVA_HOME:	jdk的安装路径   C:\install\java\jdk;
	path:	%JAVA_HOME%\bin;
	Classpath:	.;%JAVA_HOMR%\lib;
	注意：jdk 1.6 及以后可以不用配置Classpath
	
	tips：
		编译：将源文件（.java文件）翻译成二进制文件（.class文件）
		运行：执行.class文件（字节码文件）
		查看jdk版本：java -vorsion(判断jdk是否成功安装)
4、注释
	解释说明文字，不能编译运行
	1）单行注释   //注释内容
	2）多行注释  /*注释内容*/
	3）文档注释  /**注释内容*/
5、关键字
	Java中有特殊含义的单词，开发人员不能使用关键字定义包名、类名、方法名、变量名。。。。
	其中goto、const目前作为保留字（现在不用，以后可能用）
6、标识符
	用来给类、包、方法、变量等命名
	规则：必须由字母（大小写都可以）、数字、下划线、$组成
	注意事项：
		1）不能以数字开头
		2）不能使用关键字
		3）尽量不要使用$、¥和中文
		4）Java大小写严格区分
	类名：
		首字母大写，比如HelloWord，Demo
	包名：
		都是小写，如果是多级包，包与包之间用.隔开
		比如：corejava(单级包)，com.xms.basic(多级包)
	方法名：
		一个单词，小写、比如main()
		多个单词，从第二个单词开始，首字母大写，比如getMax()
	变量名：
		和方法名规则一样
	常量名：
		都是大写，如果有多个单词，用_隔开，比如PI，MAX_VALUE
7、变量
	Java是一种强类型语言，定义变量的时候，必须要声明变量的数据类型
	格式：
		数据类型 变量名 = 变量初始化值；
	说明：
		1）变量必须声明，且初始化之后，变量才可以使用、
		2）同一作用域内，变量不能重复声明
		3）超出变量作用域，变量会自动销毁
		4）变量的值可以发生更改
8、数据类型
		1）基本数据类型
			整数：byte字节型	short：短整型   int：整形	long长整型
			浮点数	：float单精度浮点型		double双精度浮点型
			字符：	char字符型
			布尔：	boolean布尔型
		2）引用数据类型
			常见的有类、String、数组
9、整型
		类型				内存空间				取值范围
		byte				  8bit					-128~127
		short			16bit					-2^15~2^15-1
		int				32bit					-2^31~2^31-1
		long				64bit					-2^63~2^63-1
		说明：
			1）Java在数学运算中，不会检查范围，在超出范围时，会发生溢出问题，造成数据损失
			2）long类型，一般用来表示时间的毫秒值，一般用来处理时间
			3）如果一个整数，没有声明类型，默认为int类型
10、浮点型
	float单精度浮点型		32bit
	double双精度浮点型	64bit
	说明：
		1）开发中，尽量避免使用浮点型进行运算，会发生精度损失
		2）如果非得使用浮点型，尽量使用double(精度比float高)
		3）float变量初始化值的时候，可以在值后面添加后缀F/f
		4)小数类型是double类型
11、布尔型
	boolean,表示状态或者条件是否成立
	值只能为true或者false
12、字符型
	Java采用Unicode编码，为每一个字符分配了一个数字编号。实际上就是一个整数。
	说明：
		1）表示方式，用单引号将一个字符括起来
		2）是一个无符号的16位整数，可以参与运算
		3）英文兼容ASCII码
		4）取值范围		0~65535
		5）支持主流国家的文字，包含两万多个汉字
	特殊字符：
		'\r':	回车      '\n'   换行           '\t'		制表   
	   '\\'   \			'\''   '			'\"'		"
		'\u4D5A'   通过Unicode找到16进制4D5A对应的字符
13、类型转换
	1）默认转换
		由小类型转换大类型
	2）强制转换
		由大类型转换小类型
	说明：
		1）类型大小排序
			byte    short   char   <   int   <   long		<		float		<		double
			boolean不参与比较
		2）byte    short   char  在参与运算的时候，会默认转换成int类型，再参与运算
		3）不涉及常量运算，只针对含有变量的运算
		4）boolean不参与类型转换
