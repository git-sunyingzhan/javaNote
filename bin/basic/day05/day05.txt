1、循环嵌套
    在循环语句内嵌套其他循环语句
    for(){
        for(){}
    }
    for(){
        while(){}
    }
    while(){
        for(){}
    }
2、方法（函数）
    完成特定功能的代码块
    格式：
        修饰符 返回值 方法名 （参数列表）{
            方法体；
            [return 返回值；]
        }
    说明：
        修饰符：当前public static
        返回值类型：
            有返回值，返回值的类型为返回值的类型
            没有返回值，返回值类型为void
        参数列表：
            参数：这个方法需要某些东西才能实现，这些东西称为参数
                参数可以是一个，也可以是多个，也可以没有参数（空参）
                如果有多个参数，参数和参数之间用逗号隔开
            return:
                结束方法，如果方法有返回值，一定要有return语句，后面加上需要返回的值
        注意：
            1）方法之间是同级关系，不可以在方法的内部嵌套其他方法
            2）方法调用不执行
            3）方法在调用的时候，不需要传递参数数据类型；
                实参和形参之间的类型不一致：
                    如果形参可以转换成实参乐境，则编译通过
                    如果形参不可以转换成实参类型，则编译不通过
3、方法重载（Overload）
    在同一类中，存在多个方法名相同而参数列表不同的方法。这种方法称为重载
    注意：方法重载和返回值类型无关
    参数列表不同：参数个数不同或者参数类型不同
4、数组（本质上就是一个对象）
    是一个用来存放相同数据类型的容器
    数组的初始化：
        1）动态初始化
            指定数组长度，由系统分配数组内的初值
            格式：
                数据类型[] 数组名 = new 数据类型[数组长度]
            初始值：
                byte、short、int、long，默认初值0
                double、float，默认0.0
                char，默认‘\u0000’
                boolean,默认false
                String，默认null（所有引用数据类型，默认都是null）
        2）静态初始化
            指定每个元素的初值，由系统分配数组长度
            格式：
                数据类型[] 数组名 = new 数据类型[]{元素1，元素2...};
                数据类型[] 数组名 = {元素1，元素2...};  (简写)
                注意：简写方式只适用于初始化的时候
